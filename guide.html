Ada Guide

What I had to find out myself

Ada is an old language and has a smaller community than other programming languages. Therefore, this will be a small gathering of good teaching material and tips, tricks and help that I had to find out for myself.

Installation on Windows

Ada has a package manager called Alire. Along with being the main package manager for Ada it also handles projects and compiling.

Installation of Alire

Website: https://alire.ada.dev/docs/

  • Go to the website and continue on to the GitHub. Download the exe.
  • Install the exe
  • Add Alire to PATH (Standard folder: C:\Program Files\Alire\bin)
  • Install msys2, either by writing alr in the command prompt or opening the Alire Powershell (found on the desktop after installation)

To install a separate compiler and a more customizable project manager, which is needed if you use VSCode, you need to get GNAT and gpr.

  • Go to any folder (I could recommend C:\Program Files, or similar) and create a folder for GNAT and gpr
  • Open the folder in the command prompt (if in C:\Program Files you need admin rights) and run:
    • $ $
    alr get gnat_native
    alr get gprbuild
